## 001 業務に役立つメール作成編

概要: 業務に役立つ生成AIの使い方は、実はチャットを少なくして、プロンプト一発で大部分を作り、その後整える方法だろう。その実践と、一発でいい感じのプロンプトを作れる、長目開発のアプリ「[はんなりプロンプトキッチン](https://github.com/hideyuki-ogawa/prompt-maker)」もご紹介。！
